# vmess 自动化安装脚本

## 快速安装
```shell
git clone https://github.com/ShawnJim/net-jumper.git
cd net-jumper
chmod +x mail.sh && ./mail.sh
```

## 验证通过环境
- Ubuntu 20.04
- CentOS 7.6

## 核心组件
- V2ray Core
- OpenResty
- Acme.sh (证书受签)
- Vnstat (监控)

## 功能模块
- 流量监控
- 邮件告警
- GFW防检测
- 自动续签证书
- 管理后台
- 订阅生成
